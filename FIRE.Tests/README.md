# FIRE.Tests

(C) 2026 by Thomas Stoll

## Purpose

`FIRE.Tests` contains automated tests for the `FIRE` API library.
The test project verifies behavior of core pipeline and supporting components.

## Test Focus Areas

- configuration validation
- metadata resolution behavior
- path generation logic
- file operation safety checks
- localization/resource fallback behavior

## Running Tests

```powershell
dotnet test FIRE.Tests/FIRE.Tests.csproj
```
