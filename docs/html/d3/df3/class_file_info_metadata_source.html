Detailed Description

Metadata source that extracts file system properties (timestamps).

FileInfoMetadataSource reads file system metadata from the Windows file system, including file creation time, modification time, and last access time. It is a lightweight, built-in metadata source that does not require external utilities.

Extracted Metadata: This source extracts the following metadata entries:

  • CreationTime: File creation timestamp (format: yyyy-MM-dd HH:mm:ss)
  • ModificationTime: File last modified timestamp
  • AccessTime: File last accessed timestamp

Error Handling: If a file does not exist or if metadata extraction fails for any reason, the source returns an empty metadata dictionary. Exceptions are caught to prevent disrupting the collection pipeline.

Registration: This source is automatically registered when a MetadataSourceRegistry is created. It is typically referenced with the source name "FILEINFO" in configuration files.
